# Distributed Machine Learning Regression Project

# Project Overview

This project implements a Distributed Regression Model using PySpark on a large-scale dataset.  
The objective of this assignment is to apply big data processing techniques and evaluate regression model performance using distributed computing.

The project was developed and executed in Google Colab.


# Dataset Information

The original dataset used in this project is large (over 1GB) and cannot be uploaded to GitHub due to size limitations.

You can download the dataset from the official source:

NYC Taxi Trip Data:
https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

On that page:

Scroll to 2019 → Yellow Taxi Trip Records (PARQUET/CSV)
Click each month link (e.g., January 2019, February 2019, … December 2019)
Files are large (~100+ MB each) and typically in Parquet format, but some may also have CSV versions.
